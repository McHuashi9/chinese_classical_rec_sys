# Changelog

## v0.1.0（2026-04-30）— 首次发布

### 古文阅读

- 文库浏览：收录 268 篇古文（教材 + 古文观止），可按标题/作者实时搜索
- 分页阅读器：基于屏幕尺寸自动分页，键盘翻页，阅读计时
- 知识追踪：阅读满 30 秒后自动更新能力值，动态学习率 + 幂律遗忘衰减

### 推荐引擎

- Gaussian i+1 模型：根据用户能力十维向量推荐难度略高于当前水平的篇目
- 匹配概率展示：每篇推荐结果标注匹配百分比
- 用户能力雷达图：10 轴 Canvas 雷达 + 进度条可视化十维能力分布

### 外观与偏好

- 亮/暗色主题：基于古籍线装书视觉的暖色调色板，一键切换，持久化保存
- 日志级别可调：设置页 ComboBox 切换 INFO / DEBUG / WARNING / ERROR

### 技术栈

C++17 · Qt 6 (QML) · SQLite3 · spdlog · Catch2 · 跨平台（Linux / Windows）
